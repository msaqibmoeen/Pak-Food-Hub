# Pak-Food-Hub
Pak Food Hub: Built a responsive food-ordering website using HTML, CSS, and JavaScript where users can browse a variety of Pakistani dishes and place orders through an interactive menu and simple ordering flow.
Pak Food Hub is a food-ordering website I built using HTML, CSS, and JavaScript. It showcases a wide range of Pakistani dishes with category-based browsing, dish details, pricing, and an interactive ordering flow. The site is fully responsive, includes a clean UI, and uses JavaScript for dynamic features like menu filtering and cart-style selection.
Live Demo: https://pakfoodhub.netlify.app/
